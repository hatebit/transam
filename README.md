# Trans America

Drive across the post apocalyptic United States.

[Play now on the browser](http://hatebit.github.io/transam/)

## Controls

- **W**: Accelerate
- **S**: Brake
- **A**: Steer left
- **D**: Steer right
- **Space**: Handbrake
- **R**: Reset car position
- **Arrow keys**: Camera movement
